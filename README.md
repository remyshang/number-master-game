# 数字大师 - 猜数字游戏

一个简单有趣的猜数字游戏，支持HTML5和微信小程序两个版本。

## 🎮 游戏特色

- **简单易懂**：猜一个1-100之间的神秘数字
- **智能提示**：遇到困难可以使用提示功能
- **统计系统**：记录游戏次数、最佳成绩等数据
- **排行榜**：查看历史最佳成绩
- **社交分享**：支持分享到微信好友和群聊
- **响应式设计**：适配各种屏幕尺寸

## 📱 版本支持

### HTML5版本
- 直接在浏览器中运行
- 支持桌面和移动设备
- 完整的游戏功能

### 微信小程序版本
- 专为微信生态优化
- 支持微信分享和社交功能
- 本地数据存储

## 🚀 快速开始

### HTML5版本
1. 直接打开 `number-master-game.html` 文件
2. 开始游戏，猜中1-100之间的数字

### 微信小程序版本
1. 使用微信开发者工具导入项目
2. 在模拟器中预览或真机调试
3. 上传发布到微信小程序平台
