# 数字大师 - 猜数字游戏

一个简单有趣的猜数字游戏，挑战你的直觉和推理能力！

## 🎮 游戏特色

- **原创玩法**：猜1-100之间的神秘数字
- **实时反馈**：即时显示猜测结果和提示
- **排行榜**：记录最佳成绩，挑战自我
- **数据统计**：追踪游戏表现和进步
- **响应式设计**：适配各种设备屏幕

## 🚀 快速开始

1. 克隆或下载此仓库
2. 直接打开 `index.html` 文件即可开始游戏
3. 或者将文件部署到GitHub Pages等平台

## 🎯 游戏规则

1. 系统随机生成一个1-100之间的数字
2. 玩家输入猜测的数字
3. 系统提示"大了"或"小了"
4. 根据提示继续猜测，直到找到正确答案
5. 记录猜测次数，挑战最少次数记录
